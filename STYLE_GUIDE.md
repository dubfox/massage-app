# Nantika Massage App - Style Guide

## Brand Colors

Based on the Nantika storefront design, the app uses the following color palette:

### Primary Colors

**Vibrant Green** (Primary Brand Color)
- Main: `#10b981` (emerald-500) - Used for primary actions, headers, and brand elements
- Light: `#34d399` (emerald-400) - Hover states, highlights
- Dark: `#059669` (emerald-600) - Active states, pressed buttons
- Background: `#d1fae5` (emerald-100) - Light backgrounds, cards

**Blue** (Secondary/Accent Color)
- Main: `#3b82f6` (blue-500) - Used for secondary actions, links, accents
- Light: `#60a5fa` (blue-400) - Hover states
- Dark: `#2563eb` (blue-600) - Active states
- Background: `#dbeafe` (blue-100) - Light backgrounds

### Neutral Colors

**White**
- Pure white: `#ffffff` - Main backgrounds, cards, content areas

**Dark Grey/Black**
- Dark: `#1f2937` (gray-800) - Text, borders, frames
- Medium: `#4b5563` (gray-600) - Secondary text
- Light: `#f3f4f6` (gray-100) - Subtle backgrounds, dividers

## Color Usage Guidelines

### Manager Interface
- **Primary Actions**: Use vibrant green (`emerald-600`, `emerald-700`)
- **Headers/Navigation**: Dark grey/black (`gray-800`, `gray-900`)
- **Backgrounds**: White with light grey accents (`gray-50`, `gray-100`)
- **Accents**: Blue for secondary actions (`blue-600`)

### Therapist Interface
- **Primary Actions**: Vibrant green (`emerald-600`, `emerald-700`)
- **Backgrounds**: Light green gradients (`emerald-50` to `emerald-100`)
- **Cards**: White with green accents

### Buttons
- **Primary**: `bg-emerald-600 hover:bg-emerald-700 text-white`
- **Secondary**: `bg-blue-600 hover:bg-blue-700 text-white`
- **Danger**: `bg-red-600 hover:bg-red-700 text-white`
- **Outline**: `border-2 border-emerald-600 text-emerald-600 hover:bg-emerald-50`

### Text Colors
- **Primary Text**: `text-gray-900` (black)
- **Secondary Text**: `text-gray-600`
- **Muted Text**: `text-gray-500`
- **Links**: `text-blue-600 hover:text-blue-700`

### Status Colors
- **Active/Success**: Green (`emerald-600`)
- **Warning**: Yellow/Amber (`amber-500`)
- **Error/Danger**: Red (`red-600`)
- **Info**: Blue (`blue-600`)

## Typography

- **Headings**: Bold, dark grey (`text-gray-900`, `font-bold`)
- **Body Text**: Regular, dark grey (`text-gray-800`)
- **Labels**: Semibold, medium grey (`text-gray-700`, `font-semibold`)

## Spacing & Layout

- **Card Padding**: `p-4` to `p-6`
- **Section Spacing**: `space-y-4` to `space-y-6`
- **Border Radius**: `rounded-lg` (8px) for cards, `rounded-xl` (12px) for modals

## Shadows

- **Cards**: `shadow-sm` or `shadow-md`
- **Modals**: `shadow-xl`
- **Elevated Elements**: `shadow-lg`
